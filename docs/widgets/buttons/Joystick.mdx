Joystick simulate a tilt knob in all directions, outputing an angle and a strength. 

## Thumbnail

![Joystick](./assets/joystick.png)


## API

| Trigger | Value |
| --- | --- |
| OnTapDown(`f32`, `f32`) | `Joystick {delta: f32, intensity: f32}` |
| OnTapUp() | `Joystick {delta: 0.0, intensity: 0.0}` |
